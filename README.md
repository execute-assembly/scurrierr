# scurrier


Scurrier is a robust and reliable windows based c2 framework

Features
  - HTTPs and SMB implants
  - file system operations
  - Screenshot
  - upload/download
  - reverse port forwarding
  - process injection
  - powershell/cmd execution
  - hot plugins via RDI
  - fully encrypted comms
  - BOF loader
  - and more
